# shader-ast-sdf2

![screenshot](https://codeberg.org/thi.ng/umbrella/media/branch/develop/assets/examples/shader-ast-sdf2d.jpg)

- [Live demo (WebGL)](http://demo.thi.ng/umbrella/shader-ast-sdf2d/)
- [Live demo (Canvas 2D)](http://demo.thi.ng/umbrella/shader-ast-sdf2d/#2d)

Basic 2D shader defined via Signed Distance Field primitives and combinators
from the
[@thi.ng/shader-ast-stdlib](https://codeberg.org/thi.ng/umbrella/src/branch/develop/packages/shader-ast-stdlib)
package, cross-compiled to GLSL and JavaScript, the latter for emulating the
shader using Canvas 2D API (`getImageData()`).

Please refer to the [example build instructions](https://codeberg.org/thi.ng/umbrella/wiki/Example-build-instructions) on the wiki.
