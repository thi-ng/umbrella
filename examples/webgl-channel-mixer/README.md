# webgl-channel-mixer

![screenshot](https://codeberg.org/thi.ng/umbrella/media/branch/develop/assets/examples/webgl-channel-mixer.jpg)

thi.ng/rdom & WebGL-based RGB channel editor. Each channel provides its own set
of RGB weights and an offset value, allowing for a wide range of effects & color
gradings...

[Live demo](http://demo.thi.ng/umbrella/webgl-channel-mixer/)

Please refer to the [example build instructions](https://codeberg.org/thi.ng/umbrella/wiki/Example-build-instructions) on the wiki.
