# scenegraph

![screenshot](https://codeberg.org/thi.ng/umbrella/media/branch/develop/assets/examples/scenegraph.png)

[Live demo](http://demo.thi.ng/umbrella/scenegraph/)

Minimal 2D scene graph with support for
[@thi.ng/geom](https://codeberg.org/thi.ng/umbrella/src/branch/develop/packages/geom)
geometry primitives, node picking and visualization via
[@thi.ng/hdom-canvas](https://codeberg.org/thi.ng/umbrella/src/branch/develop/packages/hdom-canvas).

Please refer to the [example build instructions](https://codeberg.org/thi.ng/umbrella/wiki/Example-build-instructions) on the wiki.
